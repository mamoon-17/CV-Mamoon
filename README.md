## Project Overview

This is a multi-page portfolio website built as part of a Web Technologies assignment. It serves as an online CV/Resume showcasing professional information, skills, and projects.

### Pages Included

| Page           | Description                                      |
| -------------- | ------------------------------------------------ |
| **Home**       | Header with name, title, and profile photo       |
| **About**      | About me section and professional skillset       |
| **Projects**   | Featured projects showcase                       |
| **Experience** | Education, work experience, and leadership roles |
| **Contact**    | Contact form and information                     |

## Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Styling and animations
- **Flexbox** - Responsive layout system
- **CSS Grid** - Project and card layouts
- **Responsive Design** - Mobile and desktop compatibility
